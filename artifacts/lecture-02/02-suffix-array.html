Suffix Array Explorer

Build a suffix array by sorting the suffixes of a short string, then binary-search it. Watch lo, mid, hi converge on the range of suffixes that start with your query — the read is a prefix of a range of suffixes.

Build

Each suffix begins at a different position in the string. Sort them lexicographically; the suffix array stores the starting position of each sorted suffix.
