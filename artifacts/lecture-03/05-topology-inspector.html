Assembly Topology Inspector

Real assembly graphs are messy. Three patterns dominate the mess: tips (sequencing errors), bubbles (SNPs or hetero variants), and tangles (repeats). Load a preset, apply cleanups, and watch the graph collapse toward contigs. The order of operations is forgiving — the end state is what matters.

k-mer size k = 3 (for tip-length test). Each cleanup is idempotent — running it twice is a no-op. Try applying them in canonical order (tips → bubbles → tangles) vs the reverse on "Everything": the end state is the same, the intermediate graphs are not.
