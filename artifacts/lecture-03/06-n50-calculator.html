Nx answers: “If I line contigs up longest to shortest, what length is the contig whose running total first covers x% of the assembly?” The curve is a step — Nx only changes when a new contig is crossed. The solid vertical marker sits at x = 50 (N50); the dashed marker at x = 90 (N90). In compare mode, two assemblies with the same N50 can have wildly different curves. Always report N50 alongside total length, contig count, and largest contig.
