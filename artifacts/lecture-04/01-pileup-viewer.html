Pileup Viewer

samtools mpileup hands you a wall of .,AaTt+3AAA noise. Decode it column by column, colour the mismatches, and let the pattern tell you when five "SNVs" are really one misaligned indel.

Input

Malformed lines — fix these before rendering
    0.20

    A pileup line is one column of the alignment at one reference position. ./, = base matches reference (forward/reverse strand). A C G T upper or lower = mismatch (case = strand). +Nxxx = insertion of N bases. -Nxxx = deletion. ^X starts a read (X = mapq). $ ends a read. Allele fraction (VAF) = non-reference base count / depth.
