Reading the banner

The dropped breakdown reports the first reason a row was rejected, checked in this order: FILTER → QUAL → DP (min) → DP (max) → GT. A row that fails both FILTER and QUAL is counted once under FILTER. This mirrors how production pipelines stack VCF filters: once a row is tagged, later checks don't re-tag it.
