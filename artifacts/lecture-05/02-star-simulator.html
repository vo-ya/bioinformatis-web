STAR Seed-and-Extend Simulator

STAR maps RNA-seq reads in three phases: it searches for Maximum Mappable Prefixes (MMPs), clusters nearby hits into seed groups, then extends the winning cluster with Smith-Waterman that allows a spliced jump when a canonical GT-AG motif is present. How to read this

Phase 1 finds MMPs — the longest substring starting at each read position that maps somewhere to the reference. Phase 2 groups nearby MMPs into clusters; clusters with high combined coverage survive. Phase 3 runs spliced Smith-Waterman: when a canonical GT-AG motif lines up between two MMPs, STAR takes the splice with zero intron penalty. Toggle the canonical-motif-bonus off to see what happens when the genome does not advertise splice sites.
