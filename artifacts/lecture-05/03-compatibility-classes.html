Compatibility Class Viewer

A read's compatibility class is the intersection of transcript-sets over all its k-mers. Paste a read below, watch each k-mer tag itself with the transcripts it came from, and see the running intersection shrink. If one k-mer doesn't appear anywhere, the intersection collapses to the empty set and the read is discarded. Pseudoalignment is this, done fast.
