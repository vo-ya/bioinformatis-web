Differential Accessibility Explorer

Simulate a peaks x samples count matrix for ATAC-seq, fit a negative-binomial model per peak, BH-correct, and read off power and observed FDR. This is the Lecture 6 volcano/MA workflow re-applied to chromatin peaks: the same tests, the same plots, but now the count distribution is negative binomial and the dispersion phi controls how badly replicates can go wrong.
