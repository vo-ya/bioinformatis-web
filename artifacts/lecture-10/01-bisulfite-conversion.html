Bisulfite Conversion Simulator

Toggle each CpG's methylation state on a reference, then watch bisulfite treatment + PCR produce the sequencing read. Methylated C survives; unmethylated C deaminates to U, and the PCR amplicon reads T. The aha: drop conversion efficiency below 100% and watch unmethylated cytosines spuriously stay as C — indistinguishable from real methylation in the final read.

Conversion pipeline

(1) Reference DNA click circles above to toggle methylation per CpG — filled = methylated, open = unmethylated

(2) After bisulfite treatment unmethylated C deaminates to U (probability = conversion efficiency); methylated C resists

(3) After PCR amplification U is read as T — this is the sequencing read

methylated CpG (toggle on) unmethylated CpG (toggle off) retained C (blue — reads as methylated) converted C (amber — reads as unmethylated)

Three independent reads

Each read is an independent stochastic draw at the current conversion efficiency. Per-CpG recovery compares the read's base (C → methylated call, T → unmethylated call) to the toggle state you set.
