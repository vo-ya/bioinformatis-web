Viterbi on a Graph · Path Finder

A reference graph carries a bubble — a single alternate node that lets the graph encode both the reference allele and a common variant. Align a read by Viterbi DP across graph states × read positions: at each cell V(s, t) take the max over graph predecessors. Traceback recovers the best path. Compare against Viterbi on a linearised reference where the alternate is inaccessible: variant-bearing reads pay an unavoidable mismatch. This is the heart of "genomics is Viterbi on a DAG".
