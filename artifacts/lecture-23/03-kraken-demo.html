Educational simulation. Uses a 5-genome toy database with hand-crafted overlap structure; classification accuracy curves are calibrated against published Kraken2 benchmarks but simplified. Real Kraken2 builds k-mer hash tables from RefSeq/GenBank with ~10⁹ k-mers; runtime is ~minutes on a single sample. Not a benchmark.
