Educational simulation. True β = 0.5. Generates K SNPs with random β_ZX in (0.005, 0.04); valid SNPs have β_ZY = β·β_ZX + ε; pleiotropic SNPs add α + extra noise. Methods are simplified versions of TwoSampleMR (R) — production use needs harmonised effect alleles and proper SE calibration.
